# Django Girls Tutorial

https://tutorial.djangogirls.org/en/

## Build image
```bash
make build
```

## Login container
```bash
make
```
or
```bash
make start
```


## Browsing
```bash
open http://127.0.0.1:8000/

# on firefox
# open -a Firefox http://127.0.0.1:8000/
```

## Check status
```bash
make check
```
