# Django Girls Tutorial

https://tutorial.djangogirls.org/en/

## Build image
```bash
make build
```

## Check status
```bash
make check
```

## Login container
```bash
make
```
or
```bash
make start
```

## Run server
```bash
@container
python manage.py runserver 0:8000
```

## Browsing
```bash
open http://127.0.0.1:8000/

# on firefox
# open -a Firefox http://127.0.0.1:8000/
```
